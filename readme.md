🏥 SISGEVA

Sistema de Gerenciamento de Vacinação desenvolvido para controle de pacientes e registro de vacinas.

📌 Sobre o projeto

O SISGEVA é um sistema web simples que permite:

Cadastro de pacientes

Registro de vacinas aplicadas

Visualização do histórico de vacinação

Edição de informações

O objetivo do projeto é simular um sistema básico utilizado em postos de saúde.

🚀 Funcionalidades

👤 Cadastro de pacientes

📋 Listagem de pacientes

💉 Registro de vacinas

📊 Visualização de vacinas por paciente

✏️ Edição de dados

🔄 Navegação por telas (sem recarregar página)

🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript (Vanilla)

LocalStorage (armazenamento de dados)

📂 Estrutura do projeto

/js
  storage.js   # Gerenciamento de dados
  auth.js      # Login
  pacientes.js # Lógica de pacientes
  vacinas.js   # Lógica de vacinas
  ui.js        # Controle de telas

index.html
style.css
